## 👤 Sobre mí

Soy desarrollador con años de experiencia liderando equipos técnicos en entornos de retail a gran escala. Mi día a día combina **arquitectura de software**, **decisiones de infraestructura** y **desarrollo hands-on** — porque un buen jefe de desarrollo no pierde el contacto con el código.

Me muevo con comodidad entre el backend Java más robusto y el hardware embebido más limitado. Si tiene que funcionar en producción, lo construyo para que dure.

> *"La mejor arquitectura es la que nadie nota porque nunca falla."*
